## Usage

### `Run password.exe`

Generate one password with the given options. Returns a Strong PWD.

### Available options

| Name                     | Description                                                           | Default Value |
|--------------------------|-----------------------------------------------------------------------|---------------|
| length                   | Integer, length of password.                                          | unlimited            |
| save               | Ability to save                                    | true          |   
| copy               | Ability to copy                                    | true          |
